# Barcode Scanner

Comic Shelf uses [QuaggaJS](https://github.com/ericblade/quagga2) for in-browser barcode scanning.

## Usage

1. Click **Scan Barcode** on the Add Comic page
2. Allow camera access when prompted
3. Point the camera at the comic's UPC barcode
4. The app auto-detects the barcode and looks it up in Metron

## Supported Formats

QuaggaJS supports EAN-13 and UPC-A barcodes, which are standard on comic books.
