# User Management

Admins can manage users at `/users`.

## Roles

| Role | Permissions |
|---|---|
| `ADMIN` | Full access — manage users, all comics, settings |
| `USER` | Read/write access to comics; no user management |

## Managing Users

Admins can:
- Create new users
- Change user roles
- Delete users (except themselves)

The Users page is accessible at `/users` and requires the `ADMIN` role.
